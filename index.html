About the PAD-UFES-20 Dataset

Overview

Collected from 1,373 patients in Brazil, featuring 2,298 smartphone-captured skin lesion images paired with 21 clinical features (age, sex, lesion location, symptoms like itching/bleeding/growth).

  • Labels: 6-class (BCC, SCC, Melanoma, ACK, NEV, SEK) | Binary: Cancerous vs. Non-Cancerous
  • Modalities: Images + Tabular Data
Why Ideal for Multimodal ML?

Mimics real-world diagnosis by combining visual (lesion appearance) and structured (patient history) data, enabling fusion strategies for enhanced accuracy.

Unimodal Models

CNN (ResNet-50)

Deep learning for image modality: Extracts hierarchical features from skin lesion photos using convolutional layers. Pretrained on ImageNet, fine-tuned for 6-class classification.

AUC: 0.92
Decision Tree

Tree-based classifier for tabular data: Handles categorical features like gender/region and binary symptoms (itch, bleed). Interpretable splits on clinical variables.

AUC: 0.78
XGBoost (Advanced Gradient Boosting)

Ensemble of decision trees for multimodal fusion: Boosts weak learners on combined tabular + image embeddings. Handles missing data via built-in imputation.

AUC: 0.89

Ensemble Modeling

Fusion & Ensemble Strategy

Leveraging AutoML (inspired by AutoPrognosis-M), we apply late fusion (weighted averaging of unimodal predictions), early fusion (concatenated features into XGBoost), and joint end-to-end learning. Ensemble weights optimized via Bayesian optimization for robustness.

  • Late Fusion: CNN (0.4) + DT (0.2) + XGBoost (0.4) → Final AUC: 0.94
  • Early Fusion: Tabular + CNN Embeddings → XGBoost
  • Joint Fusion: End-to-end multimodal network with attention mechanisms (added advanced feature).
